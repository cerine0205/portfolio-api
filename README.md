# Portfolio API

A Laravel REST API that powers the Portfolio System.  
This backend handles project management operations and provides the data used by the frontend application.

## Features

- Projects API – Manage portfolio projects through API endpoints
- CRUD Operations – Create, read, update, and delete projects
- Database Integration – MySQL-backed project storage
- Frontend Integration – Connected to the React frontend application

## Tech Stack

- Laravel
- PHP
- MySQL

## Project Description

This repository contains the backend for the Portfolio System.

It is responsible for:
- Storing project data in the database
- Handling API requests from the frontend
- Managing project CRUD operations
- Returning JSON responses for frontend integration

## Architecture

This project is part of a full-stack system with two repositories:

- Backend (Laravel API) – this repository
- Frontend (React) –  
https://github.com/cerine0205/portfolio-system

## API Scope

Current API functionality includes:
- Listing projects
- Creating projects
- Updating project fields
- Deleting projects
